# Founding AE OS

Building an AI-native GTM Operating System for Enterprise Account Executives.

---

# Mission

This project has three equally important goals:

1. Build a real AI-native GTM product that can eventually be used by Enterprise AEs.
2. Learn modern frontend development by building with React, Next.js and Vercel.
3. Demonstrate the mindset of a product-oriented, AI-native Founding AE.

# Project Workflow

Every development session follows the same process:

1. Review the documentation.
2. Confirm the current sprint goals.
3. Move unrelated ideas to the Parking Lot.
4. Build one small feature.
5. Verify the application still works.
6. Update the documentation before ending the sprint.
